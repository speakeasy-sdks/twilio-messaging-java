# MessagingV1ServiceFallbackMethod

The HTTP method we use to call `fallback_url`. Can be: `GET` or `POST`.


## Values

| Name     | Value    |
| -------- | -------- |
| `HEAD`   | HEAD     |
| `GET`    | GET      |
| `POST`   | POST     |
| `PATCH`  | PATCH    |
| `PUT`    | PUT      |
| `DELETE` | DELETE   |
